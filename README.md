# ResumeCraft AI

ResumeCraft AI is a Flutter application that helps users create professional resume summaries using AI.

## Features

* Generate professional resume summaries
* User-friendly Flutter UI
* AI-powered content generation
* Fast and responsive design
* Cross-platform support

## Tech Stack

* Flutter
* Dart
* OpenAI API
* HTTP Package

## Project Structure

```text
lib/
├── screens/
├── widgets/
├── services/
├── models/
└── main.dart
```

## Getting Started

### Prerequisites

* Flutter SDK
* Android Studio
* Dart SDK

### Installation

1. Clone the repository

```bash
git clone https://github.com/MS-Shweta/resumecraft_ai.git
```

1. Navigate to project folder

```bash
cd resumecraft_ai
```

1. Install dependencies

```bash
flutter pub get
```

1. Run the project

```bash
flutter run
```
